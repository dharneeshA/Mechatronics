✅ Basic CSS Programming – Styling Web Pages

In this stage of web development, I learned CSS (Cascading Style Sheets) to style and design web pages. CSS controls the look, layout, and visual appearance of HTML elements, making websites attractive and user-friendly.

✅ Skills Covered

Applying colors, fonts, and backgrounds

Styling text, paragraphs, and headings

Adding borders, padding, and margins (box model)

Working with classes and IDs

Positioning elements using display, inline, block, flex, and grid

Creating hover effects and basic animations

✅ Example Tasks Done

Styled buttons and links

Added background colors or images

Designed simple layouts with sections and divisions

Formatted tables and lists

Created responsive basic webpage sections

✅ Learning Outcome

After completing basic CSS:

I can style any simple webpage

I understand how HTML and CSS work together

I can design cleaner, more visually appealing web interfaces

This forms the foundation for advanced CSS and responsive design
