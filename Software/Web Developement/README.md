# Adra Auto Garage Website

I built this website for my father’s auto workshop.

It allows customers to:
- Book service appointments
- Submit required details with validation
- Store bookings in Firebase Firestore
- Send approval email to admin
- Send approval/rejection email to customer

## Tech Used

- HTML
- CSS
- JavaScript
- Firebase Hosting
- Firestore
- Firebase Cloud Functions
- Nodemailer (email system)

## Booking Flow

1. Customer submits form
2. Booking stored in Firestore
3. Admin receives email with approve/reject link
4. Status updated
5. Customer receives final email

Deployed using Firebase.
